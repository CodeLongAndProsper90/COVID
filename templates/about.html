About

This website generates a graph of a state's (United States) COVID-19 cases and deaths.

Made by Scott Little and Joey Lent in 3 days.

This project uses:
  • MatPlotLib
  • Flask
  • ImageIO
  • HTML5
  • CSS 3
  • Python 3
  • Repl.it
  • Data from the NY times
